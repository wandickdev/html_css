Principais Formatações

Negrito / Destaque

Negrito não semântico => NNNNNNN (com tag b)

Negrito semântico => NNNNNNN (com tag strong = forte)

Sublinhado / Ênfase

Itálico não semântico => IIIIIII (com tag i)

Itálico semântico => IIIIIII (com tag em = enfático)

Texto Marcado

Texto cor de fundo padrão (amarelo) => MMMMMMM

Texto cor de fundo azul => MMMMMMM

Texto cor de fundo verde => MMMMMMM

Texto Grande e Pequeno

Texto Grande => GGGGGGG (com tag big)

Texto Pequeno => PPPPPPP (com tag small)

Texto Deletado e Inserido

Texto Foi Deletado => DDDDDDD (com tag del)

Texto a ser Inserido => INSINSINS (com tag ins)

Texto Sublinhado

Texto Sublinhado => UUUUUUU (com tag u)

Texto Sobrescrito

Texto Sobrescrito => Xsup (com tag sup)

Texto Subscrito

Texto Subscrito => Xsub (com tag sub)
